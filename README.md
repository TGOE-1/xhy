# 80米波段无线电测向信号源项目 (RF-Fox Hunter Pro)

## 项目简介
本项目是一款**比赛级 80 米波段（3.5MHz）无线电测向信号源**，专为无线电测向运动训练与教学设计。采用 **ESP32 + SI5351 全数字架构**，结合优化的射频功放与天线匹配电路，实现了频率精准、时序标准、功率稳定的“狐狸”发射机。

> 🎯 **核心特性：**
> - **高精度：** 工作频率 3500~3600kHz，精度高达 ±1ppm。
> - **可调功率：** 发射功率 0.1~2W 连续可调。
> - **标准协议：** 支持 IARU 标准 A1~M0 共 10 个比赛呼号。
> - **智能时序：** 1 分钟自动循环发射，时序误差 < 10ms。
> - **实时监控：** 内置驻波检测（SWR）与天线匹配指示。
> - **超长续航：** 9V 电池供电，连续工作时间 > 24 小时。

---

## 核心技术

### 1. 全数字载波产生技术
- **DDS 数字合成：** 采用 SI5351 时钟发生器产生载波，彻底解决传统 LC 振荡电路的温漂问题。
- **双重校准：** 通过 ESP32 内部 RTC 与外部高精度晶振双重校准，长期频率稳定性优于 ±3Hz。
- **灵活频点：** 软件可配置任意频点，支持一键快速切换。

### 2. 标准莫尔斯电码编码
- **标准：** 严格遵循国际无线电测向联盟 (IARU) 标准。
- **速率：** 12 字/分钟（比赛级标准速率）。
- **循环：** 12 秒发射 / 48 秒静默（5台轮询模式），支持自定义呼号与周期。

### 3. 高效射频功放方案
针对不同需求优化了两套方案：
* **三极管推挽方案：** 采用 **2SD882**，线性度好、杂散小，适合 0.1~1W 输出。
* **MOS 管单端方案：** 采用 **2SK3476**，转换效率高、发热小，适合 1~2W 大功率输出。
* **滤波优化：** 增加多级低通滤波器 (LPF)，大幅压制谐波发射。

## 硬件设计

### 系统架构
1. **电源层**：9V 电池经 AMS1117 降压至 5V/3.3V 为主控供电。
2. **控制层**：ESP32-C3 处理编码逻辑、按键输入及 OLED 显示。
3. **信号层**：SI5351A 受 ESP32 控制输出 3.5MHz 弱信号。
4. **输出层**：经功率放大器（PA）和低通滤波器（LPF）后通过匹配网络发射。

### 核心模块详表
| 模块 | 核心器件 | 功能说明 |
| :--- | :--- | :--- |
| **主控单元** | ESP32-C3 | 电码编码、时序算法、交互界面 |
| **载波产生** | SI5351A | 产生高精度、低相噪载波信号 |
| **射频功放** | 2SK3476 / 2SD882 | 功率放大，提供足够的覆盖距离 |
| **电源管理** | AMS1117-5V | 数字/模拟电路分路供电，减少干扰 |
| **人机交互** | 0.96" OLED & 按键 | 实时显示频点、呼号、功率与驻波 |
| **防护电路** | 驻波检测、自恢复保险丝 | 保护功放管避免在阻抗失配时烧毁 |

## 制作与调试

### 关键调试步骤
1. **数字部分调试**：烧录固件，检查 SI5351 是否正常输出 3.5MHz 信号。
2. **静态工作点调整**：测量功放管的漏极/集电极电流，调整偏置电阻。
3. **输出功率校准**：用功率计测量输出功率，调整功放增益。
4. **天线匹配调试**：连接天线，调整匹配电感，使驻波比 < 1.5。
5. **时序校准**：校准发射周期，确保多台同步工作时不重叠。

### 常见问题排查
* **无信号输出**：检查 SI5351 I2C 地址及焊接、功放电源是否通电。
* **信号弱**：检查天线匹配网络（天线长度对 80m 波段影响极大）。
* **谐波过大**：检查 LPF 电感电容值是否偏移。
